# UserStyles & UserScripts

# Execution
> Chrome started to not support userscript manager

Install [[stylus]](https://addons.mozilla.org/en-US/firefox/addon/styl-us/) for Userstyles and [[violentmonkey]](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/) for Userscripts.
Go to the files and click on "raw".
Install them.
